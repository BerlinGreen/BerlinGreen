# BerlinGreen
Public repository for the [GreenBox](https://berlingreen.com/en/products/greenbox) and [GreenBox Plus](https://berlingreen.com/en/products/greenbox-plus) hydroponic gardens.

We currently don't have the firmware and apps for the GreenBox as Open Source but we are working on it.

Until then, we use the [Issue tracker](https://github.com/BerlinGreen/BerlinGreen/issues) to track bugs and feature requests.

# Support

If you have any support request regarding your GreenBox please reach out via support - at - berlingreen.com
